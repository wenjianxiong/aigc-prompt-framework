# 📖 案例研究 — 古庙武侠对峙（2026-05-19 产出）

## 项目概览
- **类型**：仙侠 · 真气对决
- **风格**：超写实真人电影 → engine/styles/cinematic-real.yaml（真人电影感写实） + engine/lighting/（伦勃朗布光）
- **时长**：15s · 4 个 Cut
- **人物**：女主（青衣罡气+金剑）+ 女配（红衣紫炎骨鞭）
- **场景**：古庙汉白玉台阶

## 关键经验

### 角色-武器分离
- 青衣女主三视图 → 空手，不持武器
- 红衣女配三视图 → 空手，骨鞭握柄仅悬腰间
- 紫炎骨鞭单独 9:16 纯深灰背景
- **教训**：当武器"长"在角色身上时，Seedance 容易把武器扭曲或消失

### 提示词转化
分镜描述 → Seedance 五要素提示词：
```
分镜：女主爆发金色罡气
↓
Seedance：女主（@Image1）站在古庙台阶前（@Image3），
爆发金色罡气，气浪冲天。广角仰拍。伦勃朗布光。
85mm变形宽银幕f/1.8，8K超清。
```

### 伦勃朗光在仙侠的应用
- 45°暖金主光 → 金色罡气被光照亮，形成自然高光
- 暗部压暗 → 古庙大殿沉入阴影，增强庄严感
- 三角光斑 → 女主面部形成标志性三角光（神圣感）

### 废土转仙侠——风格引擎的复用
同一个分镜模板，只切换风格引擎参数：
- 风格：超写实真人 → 废土/仙侠选不同的 engine/styles/
- 光影：伦勃朗布光 → 废土苍凉光（同位置用不同参数）
- 质感：皮肤 → 仙侠用"超写实健康皮肤" / 废土用"战损皮肤"
- 其他不变：85mm f/1.8、@引用逻辑、三视图流程
