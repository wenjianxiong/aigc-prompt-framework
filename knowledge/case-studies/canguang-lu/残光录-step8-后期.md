## 音效分层

### 音轨1：环境底噪（全片持续）
```
Cut 1-2：  绝对安静 → 极低频嗡鸣缓慢淡入
Cut 3-7：  霓虹电流嗡鸣 + 远处城市低频 + 滴水声（循环）
Cut 8：    嗡鸣音量微降，心跳声突入
Cut 黑场：  全部静音
Cut 9-14： 嗡鸣恢复 + 滴水 + 新增：极微弱的远处人声混响（暗示城市是"活的"）
Cut 15：   嗡鸣 → 经过门口时0音效 → 灯芯噼啪 → 嗡鸣恢复
Cut 16：   嗡鸣极弱 + 灯芯噼啪持续
Cut 17-18：嗡鸣 + 模糊人声（磁带倒带质感）
Cut 19：   嗡鸣 + 说完后短暂倒带声
Cut 20：   风声 + 极轻声多语人声（Ancient Greek语感，但听不清）
Cut 21：   嗡鸣突然消失 → 只剩风声（被放大）
Cut 22：   风吹 → 嗡鸣消失 → 灯芯噼啪 → 古琴滑音 → 延续至完全黑屏后3秒
```

### 音轨3：台词（口型同步）
```
台词引擎：即梦原生音频 / ElevenLabs 中文女声

林深的声音设计：
├── 内心独白 → 低语质感，近距麦克风，温暖但偏冷
│   语速：慢，每个字之间0.15s间隔
│   音高：女中音低区
├── 侧写分析（Cut 11）→ 突然加速！
│   语速：快，和专业刑警汇报一致
│   音量：微大
│   说完后恢复低语速度
└── 终场（Cut 22）→ 逐句降速
   第1句正常 → 第2句慢 → 第3句极慢 → 最后"谁还记得"几乎是在呼气

巡警的声音：
├── 巡警A → 正常男声，但有微妙的不自然——元音稍微拖长，像GPS导航
└── 巡警B → 正常男声，比巡警A"更像人"，追问那句是真实困惑

老方镜像 → 男声，但经过轻微的低通滤波 + 混响，
    听起来不像在同一个空间里
```
