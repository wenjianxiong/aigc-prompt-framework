## 省钱黄金法则

### 法则1：480p 试错，1080p 定稿
```
试错阶段全部用 480p（1积分），画面确认后再用 1080p。
可节省 80% 试错成本。
```

### 法则2：多镜头 > 逐条
```
30s短剧 = 6 Cut × 5s
- 逐条生成：6 × 5 = 30积分
- 多镜头模式：一次生成，约15-20积分
节省 33-50%
```

### 法则3：九宫格漫剧
```
3×3九宫格做一张全能参考图，一次生成15-30s段落。
vs 9格逐格生成 = 9×5 = 45积分
节省 55-65%
```

### 法则4：写对提示词再生成
```
每条失败视频 = 5积分浪费
先写好提示词，对照 pipeline/06 质量门禁逐项检查。
减少重拍率从 40% → 15%
```

### 法则5：错峰使用
```
工作日白天和周末排队最严重
尝试深夜/工作日上午生成
Pro用户有优先排队权
```
