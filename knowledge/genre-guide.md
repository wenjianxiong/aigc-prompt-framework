# 📖 AI 短剧/漫剧故事类型创作要点（14种）

每种类型标注适配风格、推荐节奏和常见陷阱。

---

## 1. 都市逆袭

```yaml
genre: 都市逆袭
core_emotion: "从底层翻身的快感"
protagonist: 普通人/被欺辱者 → 逐步显露真实身份/能力
conflict: 打脸 → 新对手出现 → 更大的打脸
style: 真人电影感写实
lighting: 伦勃朗布光(反差) / 室内柔光(日常)
pace: 极快，每15s一个打脸点
hook: "三年之期已到，恭迎XX归来！"
trap: 重复模式让人疲劳 → 每次打脸方式要不同
```

## 2. 重生复仇

```yaml
genre: 重生复仇
core_emotion: "前世冤屈今生清算"
protagonist: 被害死者 → 重生到关键时间点
conflict: 预知未来 → 改变命运 → 新对手
style: 真人电影感写实 or 国风动画（古装重生）
lighting: 前世：冷蓝调 | 今生：暖金/高反差
pace: 中快，前世闪回穿插推进
hook: "睁开眼睛，我回到了悲剧发生的前一天。"
trap: 前世今生视觉区分不清 → 用色调/画幅区分
```

## 3. 赘婿逆袭

```yaml
genre: 赘婿逆袭
core_emotion: "被轻视者的翻身"
protagonist: 低调赘婿 → 隐藏身份曝光
conflict: 家族/外人轻视 → 打脸 → 更大的敌人
style: 真人电影感写实
lighting: 前期：阴天柔光(压抑) | 后期：黄金时刻(翻身)
pace: 中快，前期铺垫3-5集后开始高频打脸
hook: "把他给我扔出去！...等等，那是..."
trap: 前期太压抑观众弃剧 → 第一集就给个小爽点
```

## 4. 甜宠/霸总

```yaml
genre: 甜宠
core_emotion: "被极致偏爱的甜蜜"
protagonist: 普通女生 × 霸道总裁/王爷
conflict: 误会 → 和解 → 更甜 → 新误会
style: 商业广告质感（光感好、画面干净）
lighting: 暖调柔光为主，逆光浪漫时刻
pace: 中慢，给甜的场景时间发酵
hook: "女人，你成功引起了我的注意。"
trap: 缺乏冲突只有甜会腻 → 适量冲突+快速和好
```

## 5. 悬疑/推理

```yaml
genre: 悬疑推理
core_emotion: "想知道真相的焦虑"
protagonist: 侦探/警察/卷入事件的普通人
conflict: 线索 → 误导 → 真相 → 反转
style: 低调光悬疑 or 黑色电影
lighting: 单一硬光源，大面暗部
pace: 中等，留时间给线索发酵
hook: "凶手不是人。"
trap: 线索太明显或太隐晦 → 用视觉暗示而非台词说明
```

## 6. 恐怖/惊悚

```yaml
genre: 恐怖惊悚
core_emotion: "可控的恐惧"
protagonist: 普通人 → 面对未知威胁
conflict: 安全 → 异常 → 威胁 → 逃脱/反击
style: 低调光悬疑 / 真人超现实
lighting: 月光/手电/屏幕光，极暗环境
pace: 前慢后快——建立安全感 (40%) → 逐步异常 (30%) → 爆发 (30%)
hook: "如果在深夜的监控里，你看到了不该存在的东西..."
trap: 全程高能 = 全程不恐怖 → 需要安全感的建立再打破
```

## 7. 科幻

```yaml
genre: 科幻
core_emotion: "对未来的惊叹或警惕"
protagonist: 科学家/普通人/AI/改造人
conflict: 技术突破 → 伦理困境 → 失控 → 抉择
style: 赛博霓虹 / 3D写实动画 / 真人超现实
lighting: 冷调科技蓝 + 暖色生命光对比
pace: 中等，给世界观的视觉展示留时间
hook: "这个世界的'人类'只剩下最后一个。"
trap: 世界观设定大于故事 → 用故事展示设定而非旁白解释
```

## 8. 废土/末世

```yaml
genre: 废土末世
core_emotion: "在荒芜中寻找希望"
protagonist: 幸存者/拾荒者
conflict: 资源稀缺 → 人类/非人威胁 → 道德困境
style: 真人电影感写实
lighting: 废土苍凉光（暖橙脏光+锈铁冷青辅光）
pace: 中慢，让荒凉感沉浸
hook: "在这片废墟中，活下去本身就是一种反抗。"
trap: 废土=单调 → 每个场景需要不同的废墟美学
```

## 9. 武侠/仙侠

```yaml
genre: 武侠仙侠
core_emotion: "快意恩仇的浪漫"
protagonist: 修士/侠客 → 从弱到强
conflict: 修炼 → 对抗 → 突破 → 宿命之敌
style: 国风动画 or 真人电影感写实
lighting: 伦勃朗布光（庄严）/ 逆光剪影（史诗）/ 月光（夜战）
pace: 中快，战斗段落快，情感段落慢
hook: "一剑破万法。"
trap: 特效堆砌 → 战斗需要有故事逻辑，不是光污染PK
```

## 10. 系统流/游戏

```yaml
genre: 系统流
core_emotion: "规则外挂的快感"
protagonist: 绑定系统的普通人
conflict: 任务 → 奖励 → 升级 → 更强任务
style: 2D扁平插画 / 日系动画（UI弹窗可漫画化处理）
lighting: 取决于世界观背景
pace: 快，任务-奖励-新任务的循环不能拖
hook: "叮！恭喜宿主绑定[XX系统]！"
trap: UI/系统弹窗的视觉化 → 推拉镜头到角色视角时展示
```

## 11. 校园/青春

```yaml
genre: 校园青春
core_emotion: "怀念或向往的少年时光"
protagonist: 学生 → 面对学业/友情/初恋
conflict: 日常 → 波动 → 和解/告别
style: 复古胶片 / 日系动画
lighting: 阴天柔光（日常）/ 黄金时刻（美好回忆）/ 蓝调时刻（伤感）
pace: 中等偏慢，留呼吸空间
hook: "那个夏天，改变了一切。"
trap: 叙事太平淡 → 每个场景需要一个情绪锚点
```

## 12. 规则怪谈

```yaml
genre: 规则怪谈
core_emotion: "打破规则的禁忌快感"
protagonist: 收到/发现奇怪规则的人
conflict: 遵守规则 → 规则被打破 → 后果 → 真相
style: 真人超现实 / 日系动画
lighting: 无影冷白光 / 月光（不自然的平静或不安）
pace: 前慢后快，规则的建立需要耐心，破裂需要爆发
hook: "请仔细阅读以下规则。违反任何一条，后果自负。"
trap: 规则太复杂 → 用视觉化方式展示规则（出现/消失的文字/符号）
```

## 13. 脑洞/沙雕

```yaml
genre: 脑洞
core_emotion: "意想不到的快乐"
protagonist: 任何人，不限制
conflict: 正常 → 荒诞 → 更荒诞
style: 欧美卡通 / 3D黏土 / 2D扁平插画
lighting: 无限制（越跳脱越好）
pace: 极快，梗的密度决定成败
hook: "如果[日常事物]突然[荒诞前提]会怎样？"
trap: 梗用完了还在水时长 → 没梗了就结束，宁可短
```

## 14. 大女主

```yaml
genre: 大女主
core_emotion: "女性力量觉醒与成长"
protagonist: 被低估/压迫的女性 → 觉醒/逆袭
conflict: 压迫 → 觉醒 → 反抗 → 独立 → 新高度
style: 真人电影感写实 / 国风动画
lighting: 前期压抑（阴天/暗调）→ 觉醒（黄金时刻/逆光剪影）→ 独立（伦勃朗）
pace: 中等，成长曲线必须有层次感
hook: "从今天起，我不再为任何人而活。"
trap: 成长太快没有说服力 → 每阶段有具体的转折事件
```
